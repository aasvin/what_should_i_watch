#Recommendations for watching Content

This file contains instructions for a Agent to make recommendations on what content the user should watch. Agent should use a json file as an input that contains the various content. 

The agent should ask the user for the following input

## Begin Inputs 
1 - How much time do they have
2 - The emotion that they are feeling right now. 

## End of Inputs


## Strict Rules - YOU MUST FOLLOW THESE RULES
1 - If it's after 10 PM, do not select content with gore or violence
2 - If the content is rated "R" or "MA" or if it's for mature audiences, then you need to confirm with the user that it's ok to show the title. There might be kids in the room 
3 - If they have more than 3 hours, then suggest a movie to watch instead of a binge worth TV show. 

## Guidelines for Movie Preference
1 - If the user has less than 30 minutes, pick a TV show. If they have more than 30 minutes, you could pick between a binge worthy TV show and a Movie. If the Movie is longer than 2 hours, then just let the user know that it will take them multiple views before finishing the movie. 

2 - This guideline helps with picking content based on emotion. 
2.1 If angry / frustrated / annoyed - prioritize content that would help calm them down and relax. 
2.2 If depressed / feeling low / sad - prioritize content that would help them smile and laugh
2.3 If bored / low on energy - content with action, high energy may be a good choice
2.4 If they are in a mood to dive deep / curious, then a cerebral documentary may be a good choice
